# Express TypeScript Boilerplate

Minimal Express + TypeScript boilerplate with ESLint (Airbnb + TypeScript) and Prettier.

## Usage

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development (hot reload):
   ```bash
   npm run dev
   ```

3. Build and run:
   ```bash
   npm run build
   npm start
   ```

4. Lint:
   ```bash
   npm run lint
   npm run lint:fix
   ```

## Files
- `src/server.ts` — main server
- `tsconfig.json` — TypeScript config
- `.eslintrc.cjs` — ESLint config
